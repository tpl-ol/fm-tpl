# tpl-ast-toml

替语言支持工具 ([TOML](https://toml.io/)).

## 特殊命令

- `tpl-ast-toml jsonrpc`

  启动 JSON-RPC 服务器 (被别的替语言工具使用).

- `tpl-ast-toml fmt`

  格式化 toml 文件 (类似 prettier).

- `tpl-ast-toml tpld-check --fmt --block=blocklist.txt`

  `.tpld.toml` 文件基本检查 (用于 CI).
